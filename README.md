# 🚖 Uber Rides Analytics Dashboard — Power BI

An interactive Power BI dashboard analyzing Uber ride data across trips, revenue, vehicles, customers, and locations — built to turn raw ride data into clear, actionable business insights.

## 📊 Dashboard Overview

The report is organized into 6 pages:

| Page | What it covers |
|------|-----------------|
| **Home** | Navigation hub for the whole report |
| **Overview** | High-level KPIs — trips, revenue, and growth at a glance |
| **Vehicle** | Breakdown of performance and demand by vehicle type |
| **Revenue** | Revenue trends, patterns, and key drivers |
| **Customer** | Customer behavior and segmentation insights |
| **Location** | Geographic distribution of rides and demand hotspots |

## 🛠️ Tools Used

- **Power BI Desktop** — data modeling, DAX, and report design
- **Power Query** — data cleaning and transformation
- **DAX** — calculated measures and KPIs

## 🔑 Key Features

- Multi-page interactive navigation
- Custom visuals and icons for a clean, branded look
- Drill-down analysis across vehicle type, customer, and location dimensions
- KPI cards for quick executive-level insights

## 📁 Files in this Repo

- `Uber.pbix` — the full Power BI report file (open in Power BI Desktop to explore interactively)
- `screenshots/` — static previews of each dashboard page

## 🚀 How to Use

1. Download `Uber.pbix`
2. Open it in [Power BI Desktop](https://powerbi.microsoft.com/desktop/) (free)
3. Explore the pages using the navigation on the Home page
